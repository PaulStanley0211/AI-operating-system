# Weekly Review Checklist
<!-- v1.0.0 -->

> Run with `/review` in Claude Code.
> Recommended: every Friday. Takes 30–60 minutes.
> A system you don't review is a system you don't trust.

---

## Phase 1 — GET CLEAR

Goal: zero inboxes. Nothing captured elsewhere that isn't in the system.

- [ ] Run `/process` on `flow/gtd/inbox.md`
- [ ] Check physical notebook or desk for uncaptured items → add to inbox
- [ ] Check email draft folder for uncaptured commitments
- [ ] Check Telegram for uncaptured items
- [ ] Inbox = 0 ✅

---

## Phase 2 — GET CURRENT

Goal: every list is accurate. No stale items.

### Projects
- [ ] Does every active project have a next action in next-actions.md?
- [ ] Are any projects actually done? → Mark complete, move to Completed section
- [ ] Are any projects stalled? → Mark on hold or delete
- [ ] Any new projects that need to be added?

### Next Actions
- [ ] Remove any items that are done
- [ ] Remove any items that are no longer relevant
- [ ] Surface items that have been sitting 3+ weeks → project or someday/maybe?
- [ ] Add context tags to any untagged items

### Waiting For
- [ ] Are any items overdue? → Follow up now
- [ ] Have any items been received? → Remove from list
- [ ] Add any new delegated items

### Someday / Maybe
- [ ] Anything worth activating this week? → Move to projects or next-actions
- [ ] Anything clearly never going to happen? → Delete

---

## Phase 3 — GET CREATIVE

Goal: catch what the task lists miss.

- [ ] Read `flow/gtd/areas.md` — for each area: anything missing from the system?
- [ ] Scan last 3 daily briefs in `tuning/briefs/` — any signals worth capturing?
- [ ] Ask yourself: "Is there anything I'm worried about that isn't in the system?"

### Trigger List (scan your mind for anything hiding)
Projects started but not tracked | Commitments to people | Waiting for responses | Unread emails | Recurring tasks not in system | Anyone I need to follow up with | Any bills, expenses, reports pending | Anything at home that needs action | Health appointments needed | Travel to plan

---

## Phase 4 — REBUILD

Goal: clear picture of the week ahead.

- [ ] Run `python flow/refresh_dashboard.py`
- [ ] Review the dashboard — does it reflect reality?
- [ ] Choose the ONE most important project for this week
- [ ] Confirm that project has a clear next action
- [ ] Optional: write a 3-sentence intention for the week

**This week's focus:** _______________

---

*Review complete. See you next Friday.*
